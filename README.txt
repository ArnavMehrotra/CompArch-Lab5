To use, pass ucode5 as an argument, followed by your user program files, 
followed by the names of all of the files in the "requried assembly files" folder.

Commands:
run x to run the simulator for x cycles.
rdump after a cycle to see the machine state.
mdump x<memlocation1> x<memlocation2> to view a range of physical memory locations.

Full documentation can be found here: http://users.ece.utexas.edu/~patt/20f.460n/
